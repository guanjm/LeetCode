# 寻找两个正序数组的中位数

## 题目描述
> 给定两个大小为m和n正序（从小到大）数组nums1和nums2.请你找出并返回这两个正序数组的中位数。
> 进阶：你能设计一个时间复杂度为0(log(m+n))的算法解决此问题吗？

## 示例1
```
输入：nums1 = [1, 3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1, 2, 3]，中位数2
```

## 示例2
```
输入：nums1 = [1, 2], nums2 = [3, 4]
输出：2.50000
解释：合并数组 = [1, 2, 3, 4]，中位数(2 + 3) / 2 =2.5
```

```
输入：nums1 = [0, 0],nums2 = [0, 0]
输出：0.00000
```
